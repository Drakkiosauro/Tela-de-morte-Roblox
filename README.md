# Tela de Morte Roblox

Sistema de tela de morte minimalista com efeito de desfoque (blur), câmera travada e barra de progresso.

## O que faz
* Deixa a tela preto e branco com desfoque forte ao morrer.
* Trava a câmera na posição da morte (impede girar).
* Mostra uma barra de carregamento sincronizada com o respawn.

## Como Instalar

1. No **Roblox Studio**, abra o painel **Explorer**.
2. Vá até a pasta **StarterPlayer** > **StarterPlayerScripts**.
3. Crie um novo **LocalScript**.
4. Cole o código.

## Configuração Importante

* Vá na pasta **Players** no Explorer.
* Nas propriedades, altere o **RespawnTime** para `6` para sincronizar com a animação.
